Content Projection

Content projection is a pattern in which you insert, or project, the content you want to use inside another component

Content Projection can be categorized into two types
  1. Single Slot Content Projection
  2. Multi Slot Content Projection
  • Single Slot Content Projection can be achieved only by using the ng-content tag.
  • Multi Slot Content Projection Can be achieved by using the selector pattern
  • Contents that already projected by using Multi Slot Content projection Will not be projected on Single Slot Content Projection.

@ContentChild Decorator

@ContentChild Decorator is used to access the reference of DOM Element or a Component or Directive from the projected content into child component class.

  • Content inside the Parent Component's ng-content is can be simply accessed by using a view child decorator.
  • @ContentChild Decorator Can't be used to access the parent components elements
